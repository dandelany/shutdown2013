About

Enigma is a search platform and API firehose for big public data. We have been monitoring interruptions in access to public data since the U.S. Government shutdown on October 1, 2013. Additionally, we are handling requests from journalists and civic media hackers for data that is no longer available via shutdown@enigma.com.


Data Sources

Furloughed employee data from agency contingency plans submitted to OMB, compiled in Google Spreadsheet.

Average employee salary data from Asbury Park Press federal salaries dataset (22MB .zip).

Archived WIC Program food cost data.

Source code and data available on Github.
